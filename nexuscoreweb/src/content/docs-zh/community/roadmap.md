# Skill Seeker 开发路线图

## 愿景
将 Skill Seeker 转变为从**任何知识源**创建 Claude AI 技能的最简单方法 - 文档网站、PDF、代码库、GitHub 仓库、Office 文档等 - 同时提供 CLI 和 MCP 接口。

## 🎯 新方法：灵活、增量式开发

**理念：** 小任务 → 选一个 → 完成 → 继续

我们现在使用**灵活的基于任务的方法**，而不是僵化的里程碑：
- 分布在 10 个类别中的 100+ 个小型独立任务
- 选择任何任务，任何顺序
- 从小开始，经常发布
- 没有截止日期，只有持续的进展

**查看：**[灵活路线图](/zh/docs/community/flexible-roadmap) 以获取完整的任务列表！

### 📋 v2.0 - 智能层（已计划）
**目标：** 智能默认值和自动配置
**时间表：** 2025 年 12 月

**功能：**

  - 检测文档框架（Docusaurus、GitBook 等）

- **质量指标：**
  - 分析生成的 SKILL.md 质量
  - 建议改进
  - 验证代码示例

- **模板：**
  - 流行框架的预构建配置
  - 社区配置共享
  - 常见文档的一键生成

**示例：**
```
用户："从 https://tailwindcss.com/docs 创建技能"
工具：自动检测 Tailwind，使用模板，在 30 秒内生成
```

## 🎨 功能想法

### 高优先级
1. **选择器自动检测** - 分析页面，建议选择器
2. **进度流式传输** - 抓取期间的实时更新
3. **配置验证 UI** - 配置质量的视觉反馈
4. **批处理** - 一次处理多个站点

### 中等优先级
5. **技能质量分数** - 评价生成的技能
6. **增强的 SKILL.md** - 更好的模板，更多示例
7. **文档框架检测** - 自动检测 Docusaurus、VuePress 等
8. **自定义类别 AI** - 使用 AI 建议类别

### 低优先级
9. **Web 仪表板** - 基于浏览器的界面
10. **技能分析** - 跟踪使用、质量指标
11. **社区配置** - 共享和发现配置
12. **插件系统** - 使用自定义抓取器扩展
